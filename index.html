Как настроить прокси (Netlify)
Из-за CORS браузер не может напрямую обращаться к Uber API. Нужна серверная функция-прокси.
Шаг 1 — Зарегистрируйтесь на netlify.com
Шаг 2 — Создайте новый сайт, загрузите файлы из архива ниже
Шаг 3 — Вставьте URL функции в поле Proxy URL выше
Шаг 4 — Добавьте переменные окружения в Netlify → Site settings → Env variables
